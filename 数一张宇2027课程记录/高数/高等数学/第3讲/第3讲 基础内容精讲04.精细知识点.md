## 一、知识点导读

1. 导数与微分计算：导数来自差商极限；复杂函数按复合、隐函数、参数方程或高阶结构分层求导。 使用场景：切线法线、变化率、函数性质和微分方程。 易错点：分段点必须算左右导数；参数方程二阶导不能直接相除。
2. 函数性质与图像：先确定定义域，再讨论奇偶、周期、单调和反函数；图像变换从点坐标变换理解。 使用场景：画图、判断方程根个数、定积分对称化。 易错点：奇偶性要求定义域关于原点对称，反函数要求一一对应。
3. 极限与连续：先判未定型，再选择等价无穷小、洛必达、泰勒或夹逼；连续要同时检查点值和极限。 使用场景：极限计算、参数确定、间断点分类、存在性证明。 易错点：等价无穷小不能随意用于和差结构；保号性是局部结论。
4. 数列极限与递推：递推数列先证有界和单调，再设极限；累加比值可考虑 Stolz；分式求和可裂项。 使用场景：递推极限、数列收敛证明、级数前置计算。 易错点：不能未证明收敛就直接设极限；极限方程多根要筛选。
5. 微分学应用：用一阶导判断单调和极值，用二阶导判断凹凸和拐点，用极限判断渐近线。 使用场景：函数图像、最值、不等式证明、曲线几何。 易错点：驻点不一定是极值点，拐点必须凹凸性改变。
6. 中值定理体系：把目标等式倒推成辅助函数的零点或导数零点，再套 Rolle、Lagrange 或 Cauchy。 使用场景：证明存在某点、证明不等式、洛必达和 Taylor 展开。 易错点：闭区间连续、开区间可导、端点条件缺一不可；导函数存在不等于导函数连续。
7. 积分应用与经济应用：先写微元，再累加；复利/现值/边际等经济应用要先把文字条件翻译成函数关系。 使用场景：几何应用、物理应用、数三复利、现值、边际和弹性。 易错点：应用题要检查变量范围、单位和适用科目；经济最值不能脱离可行区间。

2. 可导推出连续：函数在一点可导必连续；连续不一定可导，如尖点、垂直切线、振荡点。
3. 左右导数：分段点可导当且仅当左右导数都存在且相等，同时函数值要与左右极限匹配。
4. 导数几何意义：f’(x0) 是切线斜率，法线斜率为 -1/f’(x0)，需注意 f’=0 或不存在的特殊情况。
5. 微分二级结论：dy=f’(x)dx 是函数增量的线性主部，Δy=dy+o(dx)。
6. 使用场景：变化率、切线法线、单调极值、中值定理和微分方程建模。

## 五、专题二级结论详表

本表按“导数概念”继续细化，偏向做题时可直接调用的二级结论、推导入口和使用场景。

1. 左右导数：一点可导要求左右导数存在且相等；分段点必须单独检查。 使用场景：可导性判定和参数题。
2. 可导与连续：可导必连续，连续不一定可导。 使用场景：尖点、角点、垂直切线判断。
3. 切线法线：导数为切线斜率，法线斜率通常为 -1/f’。 使用场景：几何意义题。
4. 微分：dy=f’(x)dx 是 Δy 的线性主部。 使用场景：近似计算和误差估计。
5. 本讲检查清单：确认定义域、可导区间、端点、驻点、不可导点和导数变号；证明题要逐条写出连续/可导等定理条件。 使用场景：微分学计算、证明和应用题。

## 六、手动总结风格查漏补缺

本节按你 `手动总结` 的粒度补充，偏向“看到什么想到什么、公式怎么用、哪些条件不能漏”。专题：导数概念。

1. 导数定义是差商极限，不是普通斜率；必须让自变量增量趋 0。
2. 一点可导要求左右导数都存在且相等。
3. 可导必连续，连续不一定可导；尖点、角点、垂直切线都要警惕。
4. 切线斜率是 f’(x0)，法线斜率通常是 -1/f’(x0)，但 f’=0 时法线是竖直线。
5. 分段点求导先检查连续，再算左右导数；不连续必不可导。
6. 微分 dy=f’(x)dx 是线性主部，Δy 不等于 dy。
7. 高阶导数是反复求导，莱布尼茨公式用于乘积高阶导。
8. 反函数求导：若 y=f(x)，则 (f^{-1})’(y)=1/f’(x)。
