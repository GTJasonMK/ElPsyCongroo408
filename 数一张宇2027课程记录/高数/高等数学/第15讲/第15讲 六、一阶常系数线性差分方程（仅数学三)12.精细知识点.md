1. 考纲定位：一阶常系数线性差分方程为数学三内容，常用于离散经济动态模型。
2. 标准型：y_{n+1}=a y_n+b 或 y_{n+1}+p y_n=q，先求平衡解再叠加齐次解。
3. 通解结构：齐次部分通常为 C a^n，特解为常数或按右端形式设。

5. 经济场景：蛛网模型、库存、收入消费递推、离散复利。
6. 易错点：下标 n 与 n+1 不要错位，初值决定 C。

## 五、专题二级结论详表

本表按“一阶常系数线性差分方程”继续细化，偏向做题时可直接调用的二级结论、推导入口和使用场景。

1. 标准型识别：一阶常系数线性差分方程通常整理为 y_{n+1}=a y_n+b 或 y_{n+1}+p y_n=q。 使用场景：把经济递推、库存递推、蛛网模型统一成可解形式。
2. 平衡解：令 y_{n+1}=y_n=y*，先求稳定状态；再令 z_n=y_n-y* 化成齐次递推。 使用场景：快速得到通解结构并判断长期趋势。
3. 通解结构：齐次部分为 C a^n，非齐次常数项由平衡解给出；初值 y_0 或 y_1 决定 C。 使用场景：求特解、预测第 n 期状态。
4. 稳定性：|a|<1 时趋向平衡解，|a|>1 通常发散，a<0 时可能出现交替振荡。 使用场景：判断经济系统是否收敛、蛛网模型是否稳定。
5. 本讲检查清单：先识别方程类型，再求通解，代入初值定常数，最后把解代回原方程并检查定义区间或稳定条件。 使用场景：微分方程和差分方程完整作答。

2. 标准型 y_{n+1}=ay_n+b。
