## 一、知识点导读

1. 逻辑命题与证明方法：先把自然语言翻译成量词和蕴含关系，再选择直接证明、逆否证明或反证法。 使用场景：证明题、存在性判断、恒成立/有解类问题。 易错点：否定命题时要同时改变量词和结论，不能只把结论取反。
2. 解析式分类与代数变形：把复杂表达式拆成整式、分式、根式等基本结构，再决定因式分解、有理化或换元。 使用场景：极限化简、积分前化简、方程不等式整理。 易错点：约分和换元前要检查定义域，根式开方后常出现绝对值。
3. 二次函数、方程与不等式：用开口方向、判别式、对称轴、端点函数值把代数条件翻译成图像位置。 使用场景：含参数二次方程根分布、恒成立、最值问题。 易错点：乘除含参数不等式前必须判断符号；根的位置问题不要漏判别式。
4. 函数性质与图像：先确定定义域，再讨论奇偶、周期、单调和反函数；图像变换从点坐标变换理解。 使用场景：画图、判断方程根个数、定积分对称化。 易错点：奇偶性要求定义域关于原点对称，反函数要求一一对应。
5. 数列极限与递推：递推数列先证有界和单调，再设极限；累加比值可考虑 Stolz；分式求和可裂项。 使用场景：递推极限、数列收敛证明、级数前置计算。 易错点：不能未证明收敛就直接设极限；极限方程多根要筛选。
6. 极限与连续：先判未定型，再选择等价无穷小、洛必达、泰勒或夹逼；连续要同时检查点值和极限。 使用场景：极限计算、参数确定、间断点分类、存在性证明。 易错点：等价无穷小不能随意用于和差结构；保号性是局部结论。
7. 微分学应用：用一阶导判断单调和极值，用二阶导判断凹凸和拐点，用极限判断渐近线。 使用场景：函数图像、最值、不等式证明、曲线几何。 易错点：驻点不一定是极值点，拐点必须凹凸性改变。
8. 中值定理体系：把目标等式倒推成辅助函数的零点或导数零点，再套 Rolle、Lagrange 或 Cauchy。 使用场景：证明存在某点、证明不等式、洛必达和 Taylor 展开。 易错点：闭区间连续、开区间可导、端点条件缺一不可；导函数存在不等于导函数连续。

## 四、复查补充：考纲/二级结论/常用推论

本节已按文件标题和课堂原文主题复查，专题定位为：数列基础。补充内容只增加考纲常见要求、二级结论、推导入口和使用场景，不删改课堂原文。

1. 考纲衔接：数列极限是函数极限的离散版本，重点在单调有界、递推关系和常见放缩。
2. 单调有界二级结论：单调递增且有上界必收敛，单调递减且有下界必收敛；先证单调再找界。
3. 递推数列：若极限存在，可在递推式两边取极限求候选值，但还必须证明极限确实存在。
4. 夹逼与放缩：含根式、阶乘、幂次时常用放缩到已知极限；注意放缩方向要与目标一致。
5. 常见比较：n! 增长快于 a^n，a^n 快于 n^k，ln n 慢于 n^α。
6. 易错点：通项趋于 0 不代表级数收敛；数列有界也不代表收敛。

本表按“数列基础”继续细化，偏向做题时可直接调用的二级结论、推导入口和使用场景。

1. 存在性：双侧极限看左右一致，数列看所有子列一致。 使用场景：判断极限是否存在。
2. 等价与Taylor：乘除可用等价，和差相消用 Taylor。 使用场景：极限计算。
3. 夹逼：有界量乘无穷小为无穷小。 使用场景：振荡和三角极限。
4. 连续三要素：点值、极限、相等三者缺一不可。 使用场景：分段函数和参数题。
5. 本讲检查清单：确认趋近方式、左右极限、是否在点有定义、等价替换是否合法；数列题还要检查单调有界或子列。 使用场景：极限连续和数列极限题。

## 六、手动总结风格查漏补缺

本节按你 `手动总结` 的粒度补充，偏向“看到什么想到什么、公式怎么用、哪些条件不能漏”。专题：数列基础。

1. 数列是定义在正整数集上的函数；单调性比较 a_{n+1}-a_n 或 a_{n+1}/a_n。
2. 等差数列通项 a_n=a_1+(n-1)d，前 n 项和可首尾配对。
3. 等比数列通项 a_n=a_1q^{n-1}，求和时注意 q=1 单独讨论。
4. 单调有界必收敛；只说有界或只说单调都不够。
5. 递推数列不能一上来设极限，必须先证明极限存在。
6. 递推极限方程可能有多个根，要结合单调性、范围或初值筛选。
7. n! 比 a^n 长得快，a^n 比 n^k 快，n^k 比 ln n 快。
8. 裂项求和要写出前后抵消项，别只写“显然”。
