## 一、知识点导读

1. 逻辑命题与证明方法：先把自然语言翻译成量词和蕴含关系，再选择直接证明、逆否证明或反证法。 使用场景：证明题、存在性判断、恒成立/有解类问题。 易错点：否定命题时要同时改变量词和结论，不能只把结论取反。
2. 解析式分类与代数变形：把复杂表达式拆成整式、分式、根式等基本结构，再决定因式分解、有理化或换元。 使用场景：极限化简、积分前化简、方程不等式整理。 易错点：约分和换元前要检查定义域，根式开方后常出现绝对值。
3. 二次函数、方程与不等式：用开口方向、判别式、对称轴、端点函数值把代数条件翻译成图像位置。 使用场景：含参数二次方程根分布、恒成立、最值问题。 易错点：乘除含参数不等式前必须判断符号；根的位置问题不要漏判别式。
4. 函数性质与图像：先确定定义域，再讨论奇偶、周期、单调和反函数；图像变换从点坐标变换理解。 使用场景：画图、判断方程根个数、定积分对称化。 易错点：奇偶性要求定义域关于原点对称，反函数要求一一对应。
5. 数列极限与递推：递推数列先证有界和单调，再设极限；累加比值可考虑 Stolz；分式求和可裂项。 使用场景：递推极限、数列收敛证明、级数前置计算。 易错点：不能未证明收敛就直接设极限；极限方程多根要筛选。
6. 极限与连续：先判未定型，再选择等价无穷小、洛必达、泰勒或夹逼；连续要同时检查点值和极限。 使用场景：极限计算、参数确定、间断点分类、存在性证明。 易错点：等价无穷小不能随意用于和差结构；保号性是局部结论。
7. 导数与微分计算：导数来自差商极限；复杂函数按复合、隐函数、参数方程或高阶结构分层求导。 使用场景：切线法线、变化率、函数性质和微分方程。 易错点：分段点必须算左右导数；参数方程二阶导不能直接相除。
8. 中值定理体系：把目标等式倒推成辅助函数的零点或导数零点，再套 Rolle、Lagrange 或 Cauchy。 使用场景：证明存在某点、证明不等式、洛必达和 Taylor 展开。 易错点：闭区间连续、开区间可导、端点条件缺一不可；导函数存在不等于导函数连续。

## 四、复查补充：考纲/二级结论/常用推论

本节已按文件标题和课堂原文主题复查，专题定位为：方程与不等式。补充内容只增加考纲常见要求、二级结论、推导入口和使用场景，不删改课堂原文。

1. 考纲衔接：方程与不等式服务根的个数、区间划分、恒成立和参数范围题。
2. 二次函数二级结论：判别式控制实根个数，顶点给最值，开口方向决定大于/小于零的区间。
3. 分式不等式：先移到一边通分，再按零点和无定义点分区间列表；不能直接交叉相乘忽略符号。
4. 绝对值不等式：|x-a|<r 表示距离小于 r；|A|>|B| 可平方，但要确认平方后的等价条件。
5. 恒成立处理：含参数不等式常转为参数 >= 函数最大值或参数 <= 函数最小值。
6. 易错点：端点能否取到由原式定义域和不等号严格性共同决定。

## 六、手动总结风格查漏补缺

本节按你 `手动总结` 的粒度补充，偏向“看到什么想到什么、公式怎么用、哪些条件不能漏”。专题：方程与不等式。

1. 一元二次方程先看 a 是否为 0；若 a=0 就退化为一次方程，不能直接套判别式。
2. 判别式 Δ>0 两个不等实根，Δ=0 两个相等实根，Δ<0 实数域无根。
3. 根与系数关系：二次方程 x1+x2=-b/a，x1x2=c/a；n 次方程要注意符号交替。
4. 根的位置问题看开口方向、判别式、对称轴、端点函数值，四件事缺一不可。
5. 不等式两边同乘负数必须变号；同乘含参数式前必须先判断它的符号。
6. 均值不等式调几算方：调和 <= 几何 <= 算术 <= 均方根，等号通常在各量相等时成立。
7. 三角不等式：||a|-|b|| <= |a+b| <= |a|+|b|；绝对值先想到距离。
8. 柯西不等式看到“平方和、乘积和、向量点积”就要主动联想。
9. 恒成立问题通常转成函数最值；有解问题通常转成值域相交。
10. 出现 a/b 不好放缩时，可把它看作 a*(1/b)，再考虑均值或柯西。
11. 每用一个不等式都要检查等号成立条件，否则最值题容易漏边界。
